🚀 Anonymous Feedback App

Quick Start Guide - Choose Your Method

✅ Good News! Your app is fully created and ready to run!
⚠️ Note: You need Maven installed for the backend. For now, let's start with just the frontend to see the UI!

🎯 EASIEST METHOD (Just Frontend UI)

1 Open File Explorer
Press Windows Key + E
2 Navigate to Your Project
Go to: D:\GitHubProjects\anonymous-feedback-app
3 Double-Click This File
Find and double-click: QUICK-START.bat
(This will start the frontend and open your browser automatically!)
4 Wait for Browser
Browser will automatically open to: http://localhost:3000
First time takes 2-3 minutes (installing packages)

💻 COMMAND LINE METHOD

1 Open PowerShell or Command Prompt
Press Windows Key + R, type powershell, press Enter
2 Copy and Paste These Commands
cd D:\GitHubProjects\anonymous-feedback-app\frontend
npm install
npm start
3 Visit in Browser
Open browser and go to: http://localhost:3000

📋 What You'll See

✅ Beautiful landing page with gradient design
✅ Register/Login pages
✅ Feature showcase
✅ Fully responsive design

Note: Backend features (creating forms, submitting feedback) won't work until you install Maven and start the backend. But you can see the entire beautiful UI!

🛠️ To Install Maven (For Full Functionality)

1 Run as Administrator
Right-click PowerShell → "Run as Administrator"
2 Install Maven
cd D:\GitHubProjects\anonymous-feedback-app
.\INSTALL-MAVEN.bat
3 After Installation
Close all terminals, open new one, and run: START.bat
